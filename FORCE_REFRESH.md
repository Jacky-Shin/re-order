# 强制刷新浏览器缓存

如果页面仍然显示旧内容（中文文本或¥符号），请按照以下步骤操作：

## 🚀 快速方法（推荐）

### 方法1: 强制刷新快捷键
- **Windows**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

### 方法2: 开发者工具强制刷新
1. 按 `F12` 打开开发者工具
2. **Chrome/Edge**: 右键点击刷新按钮 → 选择"清空缓存并硬性重新加载"
3. **Firefox**: 网络标签 → 勾选"禁用缓存" → 刷新

## 🔧 详细步骤

### Chrome/Edge 浏览器：
1. 按 `F12` 打开开发者工具
2. 在开发者工具打开的情况下，右键点击浏览器地址栏的刷新按钮
3. 选择"清空缓存并硬性重新加载"
4. 或者：按 `Ctrl + Shift + Delete` → 选择"缓存的图片和文件" → 点击"清除数据"

### Firefox 浏览器：
1. 按 `F12` 打开开发者工具
2. 点击"网络"标签
3. 勾选"禁用缓存"复选框
4. 按 `F5` 刷新页面

### Safari 浏览器：
1. 按 `Cmd + Option + E` 清空缓存
2. 按 `Cmd + R` 刷新页面
3. 或者：Safari菜单 → 偏好设置 → 高级 → 勾选"在菜单栏中显示开发菜单" → 开发 → 清空缓存

## 📱 移动设备

### iPhone/iPad (Safari):
1. 设置 → Safari → 清除历史记录和网站数据
2. 或者：长按刷新按钮 → 选择"重新载入页面（不使用缓存）"

### Android (Chrome):
1. Chrome菜单 → 设置 → 隐私和安全 → 清除浏览数据
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

## ⚠️ 如果仍然没有更新

1. **等待自动更新**: 系统会在2分钟内自动检测新版本并刷新
2. **检查版本号**: 打开浏览器控制台（F12），查看是否有版本更新提示
3. **清除所有站点数据**: 
   - Chrome: 地址栏左侧锁图标 → 网站设置 → 清除数据
   - Firefox: 地址栏左侧图标 → 更多信息 → 清除数据

## 🔍 验证更新

更新后，你应该看到：
- ✅ 搜索框显示 "Buscar productos..."（而不是"搜索商品..."）
- ✅ 价格显示为 "$200.00"（而不是"¥200.00"）
- ✅ 所有中文文本都变成了西班牙语

如果按照上述步骤操作后仍然没有更新，请检查：
1. Vercel 部署是否完成（查看部署状态）
2. 是否访问的是正确的 URL
3. 是否有其他浏览器扩展干扰
