# 商家后台使用指南

## 进入商家后台的方式

### 方式一：从菜单页面进入（推荐）
1. 访问菜单页面：`http://localhost:3000/menu`
2. 点击页面右上角的**设置图标**（⚙️）
3. 自动跳转到商家后台首页

### 方式二：直接访问URL
直接在浏览器地址栏输入：
```
http://localhost:3000/admin
```

### 方式三：从扫码页面进入
1. 访问扫码页面：`http://localhost:3000`
2. 点击"开始点餐"进入菜单
3. 点击右上角设置图标进入后台

## 后台功能模块

### 1. 数据统计面板 (`/admin`)
- **今日订单**：显示今天的订单数量
- **待处理订单**：显示待支付的订单数
- **今日收入**：显示今天的总收入
- **商品总数**：显示当前商品数量
- **订单状态概览**：待支付、制作中、待取餐、总订单

### 2. 商品管理 (`/admin/menu`)
- **添加商品**：点击"添加商品"按钮
- **编辑商品**：点击商品行的编辑图标
- **删除商品**：点击商品行的删除图标
- **上架/下架**：在编辑商品时设置

**操作步骤：**
1. 进入后台首页
2. 点击"商品管理"卡片
3. 点击"添加商品"按钮
4. 填写商品信息：
   - 商品名称（必填）
   - 英文名称（可选）
   - 分类（必填，如：经典咖啡、星冰乐、茶饮等）
   - 价格（必填）
   - 图片URL（可选）
   - 描述（可选）
   - 上架状态（勾选表示上架）
5. 点击"添加"保存

### 3. 订单管理 (`/admin/orders`)
- **查看订单**：显示所有订单列表
- **筛选订单**：按状态筛选（全部、待支付、制作中、待取餐、已完成）
- **更新订单状态**：
  - 待支付 → 点击"开始制作"
  - 制作中 → 点击"制作完成"
  - 待取餐 → 点击"订单完成"
  - 取消订单 → 点击"取消订单"

**操作步骤：**
1. 进入后台首页
2. 点击"订单管理"卡片
3. 使用顶部筛选按钮查看不同状态的订单
4. 点击订单卡片中的状态按钮更新订单状态

### 4. 支付记录 (`/admin/payments`)
- **查看支付记录**：显示所有支付记录
- **查看总收入**：页面顶部显示总收入统计
- **支付信息**：交易号、订单号、支付方式、金额、状态、支付时间

### 5. 收款账户管理 (`/merchant/settings`)
- **添加收款账户**：绑定银行卡信息
- **编辑账户**：修改账户信息
- **删除账户**：删除不需要的账户
- **设置默认账户**：设置默认收款账户

## 快速导航

| 功能 | URL路径 | 说明 |
|------|---------|------|
| 后台首页 | `/admin` | 数据统计和快捷入口 |
| 商品管理 | `/admin/menu` | 管理商品信息 |
| 订单管理 | `/admin/orders` | 处理订单 |
| 支付记录 | `/admin/payments` | 查看支付记录 |
| 收款账户 | `/merchant/settings` | 管理收款账户 |

## 常见问题

### Q: 找不到设置图标？
A: 设置图标在菜单页面右上角，购物车图标的左边。如果看不到，请刷新页面。

### Q: 添加商品后前台看不到？
A: 检查商品的上架状态是否勾选，只有上架的商品才会在前台显示。

### Q: 如何修改商品价格？
A: 在商品管理页面，点击商品行的编辑图标，修改价格后点击"更新"。

### Q: 订单状态如何更新？
A: 在订单管理页面，找到对应订单，点击状态按钮（如"开始制作"、"制作完成"等）。

### Q: 如何查看今天的收入？
A: 后台首页的"今日收入"卡片显示今天的总收入，也可以在支付记录页面查看详细记录。

## 提示

- 后台数据会实时更新，无需刷新页面
- 删除商品前请确认，删除后无法恢复
- 订单状态更新后，顾客端会自动看到状态变化
- 建议定期查看支付记录，核对收入情况
