# 自动化打包方案

## 🎯 我已经为您准备的自动化工具

由于我无法直接访问Mac环境，我为您创建了**完全自动化**的打包脚本，您只需要在Mac（或云Mac）上运行一次即可。

## 📦 自动化脚本

### `client/auto-build-ios.sh`

这个脚本会自动完成：
1. ✅ 环境检查（确保在Mac上，Xcode已安装）
2. ✅ 安装npm依赖
3. ✅ 构建前端代码
4. ✅ 同步到iOS项目
5. ✅ 打开Xcode

**您只需要**：
- 在Xcode中配置签名（选择开发团队）
- Product → Archive
- Distribute App → 导出.ipa

## 🚀 使用步骤

### 在Mac或云Mac上：

```bash
# 1. 进入项目目录
cd client

# 2. 给脚本执行权限
chmod +x auto-build-ios.sh

# 3. 运行脚本
./auto-build-ios.sh
```

脚本会自动完成所有准备工作，然后打开Xcode，您只需要完成最后的打包步骤。

## 📱 如果没有Mac

### 最快方案：AWS Mac Instances

1. **注册AWS账号**（免费注册）
2. **启动Mac实例**：
   - 登录AWS控制台
   - 搜索 "EC2 Mac"
   - 启动一个Mac实例（选择最小的即可）
   - 等待10-15分钟启动完成

3. **连接到Mac**：
   - 使用SSH或远程桌面
   - 上传项目文件

4. **运行脚本**：
   ```bash
   cd client
   ./auto-build-ios.sh
   ```

5. **在Xcode中打包**（按照脚本提示）

6. **下载.ipa文件**

7. **关闭实例**（节省费用）

**总成本**：约$2-5（使用2-3小时）
**总时间**：约1-2小时
