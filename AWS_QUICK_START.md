# AWS Mac实例快速开始

## 🌐 AWS官方网站

### 主网站
**https://aws.amazon.com/**

### 注册账号
**https://aws.amazon.com/cn/free/**

### EC2控制台（启动Mac实例的地方）
**https://console.aws.amazon.com/ec2/**

### Mac实例文档
**https://aws.amazon.com/ec2/instance-types/mac/**

## 🚀 快速开始步骤

### 第一步：注册AWS账号
1. 访问：https://aws.amazon.com/cn/free/
2. 点击"创建免费账户"
3. 填写信息并完成验证
4. 需要信用卡（但免费套餐不会收费，除非超出限制）

### 第二步：登录控制台
1. 访问：https://console.aws.amazon.com/
2. 使用刚才注册的账号登录

### 第三步：启动Mac实例
1. 在控制台搜索框输入 "EC2"
2. 点击进入EC2服务
3. 点击左侧 "实例" → "启动实例"
4. 在搜索框输入 "mac" 或 "macOS"
5. 选择 "macOS" 实例类型
6. 选择最小的配置（mac1.metal）即可
7. 配置其他设置（使用默认即可）
8. 点击 "启动实例"

### 第四步：连接到Mac
等待实例启动完成后（约10-15分钟）：
1. 在EC2控制台找到您的实例
2. 点击实例ID进入详情
3. 查看连接信息
4. 使用SSH或远程桌面连接

## 💰 费用说明

- **按小时计费**：约$1-2/小时
- **最小计费单位**：24小时（Mac实例有最小计费周期）
- **首次使用**：建议使用2-3小时完成打包
- **总费用**：约$24-48（因为最小计费24小时）

⚠️ **重要**：Mac实例有24小时最小计费周期，即使只使用1小时也会按24小时收费。

## 📝 替代方案（如果AWS太贵）

如果AWS的24小时最小计费太贵，可以考虑：

### 1. MacinCloud（推荐）
- 网址：https://www.macincloud.com/
- 价格：$20-30/月
- 优点：按月付费，可以随时使用

### 2. MacStadium
- 网址：https://www.macstadium.com/
- 价格：$99/月起
- 优点：专业、稳定

## 🎯 推荐方案

**如果只需要打包一次**：
- 使用MacinCloud（$20-30/月，可以随时使用）

**如果需要长期使用**：
- 考虑购买Mac mini（一次性投资）

**如果预算充足**：
- 使用AWS（按需付费，灵活）
