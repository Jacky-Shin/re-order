# 免费解决方案（不需要信用卡）

## 🎯 最佳方案：找有Mac的朋友帮忙（完全免费）⭐

这是最简单、最经济的方案，完全不需要任何费用。

## 🚀 完整步骤

### 第一步：准备项目文件

您需要将项目发送给有Mac的朋友。有两种方式：

#### 方式1：使用Git（推荐）
如果项目在Git仓库中：
1. 将代码推送到GitHub/GitLab等
2. 告诉朋友仓库地址
3. 朋友克隆项目

#### 方式2：压缩文件
1. 将整个 `re_order` 文件夹压缩成zip
2. 通过网盘（百度网盘、Google Drive等）分享给朋友
3. 或直接通过U盘/微信发送

### 第二步：朋友在Mac上操作

#### 1. 解压/克隆项目
```bash
cd ~/Downloads  # 或项目所在位置
unzip re_order.zip  # 如果是压缩文件
# 或
git clone <your-repo-url>  # 如果是Git仓库
cd re_order
```

#### 2. 安装依赖
```bash
cd client
npm install
```

#### 3. 运行自动打包脚本
```bash
chmod +x auto-build-ios.sh
./auto-build-ios.sh
```

脚本会自动：
- ✅ 检查环境
- ✅ 构建前端
- ✅ 同步到iOS
- ✅ 打开Xcode

#### 4. 在Xcode中完成打包

按照脚本提示：
1. **配置签名**：
   - 选择左侧的 "App" 项目
   - 选择 "App" target
   - 打开 "Signing & Capabilities" 标签
   - 勾选 "Automatically manage signing"
   - 选择开发团队（朋友的Apple ID）

2. **选择设备**：
   - 在顶部工具栏选择 "Any iPad"

3. **构建Archive**：
   - 菜单：`Product` → `Archive`
   - 等待构建完成（约10-20分钟）

4. **导出.ipa文件**：
   - 在Organizer中选择Archive
   - 点击 "Distribute App"
   - 选择 "Ad Hoc" 或 "Development"
   - 导出.ipa文件到桌面

#### 5. 发送.ipa文件给您

朋友将导出的.ipa文件通过网盘或微信发送给您。

### 第三步：您安装到iPad

使用3uTools：
1. 下载：https://www.3u.com/
2. 连接iPad到电脑
3. 应用 → 安装 → 选择.ipa文件
4. 在iPad上信任开发者
