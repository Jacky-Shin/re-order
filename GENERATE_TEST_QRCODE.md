# 生成测试二维码 - 快速指南

## 🚀 方法1：在应用内生成（最简单）

### 步骤：

1. **访问二维码生成页面**
   ```
   https://你的应用域名.vercel.app/qrcode
   ```
   或本地运行：
   ```
   http://localhost:3000/qrcode
   ```

2. **确认或修改基础URL**
   - 页面会自动检测当前URL
   - 如果不对，手动输入正确的Vercel部署URL

3. **可选：输入桌号**
   - 例如：`1`、`A1`、`桌1` 等
   - 不输入也可以，会生成通用二维码

4. **下载或打印二维码**
   - 点击"下载二维码"保存PNG图片
   - 或点击"打印二维码"直接打印

---

## 🛠️ 方法2：使用命令行工具

### 步骤：

1. **获取你的Vercel部署URL**
   - 访问 Vercel Dashboard
   - 找到你的项目
   - 复制部署URL（例如：`https://re-order-xxx.vercel.app`）

2. **运行生成命令**
   ```bash
   # 在项目根目录运行
   node generate-qrcode.js https://你的应用域名.vercel.app
   ```

3. **找到生成的二维码**
   - 文件位置：`qrcode-output/starbucks-menu-qrcode.png`
   - 用手机扫描即可测试

---

## 📱 测试步骤

1. **用手机扫描二维码**
   - 使用微信、支付宝、或任何扫码应用
   - 或直接用手机相机扫描

2. **验证跳转**
   - 应该跳转到菜单页面（`/menu`）
   - 如果输入了桌号，URL应该包含 `?table=桌号`

3. **测试点餐流程**
   - 浏览菜单
   - 添加商品到购物车
   - 提交订单
   - 选择支付方式
   - 测试银行卡支付

---

## 💡 提示

- **本地测试**：如果手机和电脑在同一WiFi，可以使用电脑IP地址
  ```
  http://192.168.1.100:3000/qrcode
  ```
  或直接生成：
  ```
  node generate-qrcode.js http://192.168.1.100:3000
  ```

- **生产环境**：使用Vercel部署URL
  ```
  https://你的应用域名.vercel.app/qrcode
  ```

- **桌号功能**：如果输入了桌号，扫码后会自动关联到该桌号，方便商家管理

## ❓ 常见问题

**Q: 二维码扫描后打不开？**
- 检查URL是否正确
- 确认应用已部署并可以访问
- 检查网络连接

**Q: 如何找到Vercel部署URL？**
- 访问 Vercel Dashboard
- 点击项目
- 在 Deployments 页面可以看到部署URL
- 或点击项目设置 → Domains 查看域名

**Q: 可以生成多个不同桌号的二维码吗？**
- 可以！在二维码生成页面输入不同桌号
- 每次都会生成新的二维码
- 下载后可以分别打印到不同餐桌
