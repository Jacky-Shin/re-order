# 清除浏览器缓存指南

如果看到页面没有更新（仍然显示旧的中文文本或¥符号），请按照以下步骤清除缓存：

## 方法1: 强制刷新（推荐）

### Windows/Linux:
- **Chrome/Edge**: 按 `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Firefox**: 按 `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Safari**: 按 `Cmd + Shift + R`

### Mac:
- **Chrome/Edge**: 按 `Cmd + Shift + R`
- **Firefox**: 按 `Cmd + Shift + R`
- **Safari**: 按 `Cmd + Option + E` 然后刷新

## 方法2: 清除浏览器缓存

### Chrome/Edge:
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### Firefox:
1. 按 `F12` 打开开发者工具
2. 点击网络标签
3. 勾选"禁用缓存"
4. 刷新页面

### Safari:
1. 按 `Cmd + Option + E` 清空缓存
2. 按 `Cmd + R` 刷新页面

## 方法3: 清除所有站点数据

### Chrome/Edge:
1. 点击地址栏左侧的锁图标
2. 点击"网站设置"
3. 点击"清除数据"
4. 刷新页面

### Firefox:
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 选择"缓存"
3. 点击"立即清除"
4. 刷新页面

## 方法4: 使用无痕模式

打开无痕/隐私浏览窗口，访问网站，这样可以绕过缓存。

## 方法5: 等待自动更新

系统会自动检测新版本（1.0.5），如果检测到新版本，会在页面顶部显示蓝色更新横幅，并自动刷新页面。

如果2分钟后仍然没有自动更新，请使用上述方法手动清除缓存。
