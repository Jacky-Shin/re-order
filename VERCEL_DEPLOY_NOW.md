## 🎯 现在部署到Vercel（5分钟）

### 第一步：访问Vercel

1. **打开浏览器**
2. **访问**：https://vercel.com/
3. **点击右上角的 "Sign Up"** 或 **"Log In"**

### 第二步：使用GitHub登录

1. **选择 "Continue with GitHub"**
2. **授权Vercel访问您的GitHub账号**
3. **完成登录**

### 第三步：导入项目

1. **登录后，点击 "Add New Project"** 或 **"Import Project"**
2. **在仓库列表中找到 `re-order`**
3. **点击 "Import"** 旁边的按钮

### 第四步：配置项目（重要！）

在配置页面，请设置：

1. **Framework Preset**
   - 选择：**"Vite"**（会自动检测）

2. **Root Directory**
   - 点击 **"Edit"**
   - 输入：`client`
   - 点击 **"Continue"**

3. **Build and Output Settings**
   - Build Command: `npm run build`（应该自动填充）
   - Output Directory: `dist`（应该自动填充）
   - Install Command: `npm install`（应该自动填充）

4. **Environment Variables**
   - 暂时不需要，直接跳过

5. **点击 "Deploy"**

### 第五步：等待部署

- 部署过程约2-3分钟
- 您可以看到构建日志
- 等待完成

### 第六步：获取应用URL

部署完成后，Vercel会显示：
- ✅ **部署成功**
- 🌐 **应用URL**（如：`re-order.vercel.app` 或 `re-order-xxx.vercel.app`）

**请复制这个URL！**

## 📱 在iPad上使用

### 添加到主屏幕：

1. **在iPad上打开Safari**
2. **访问您的应用URL**（刚才复制的）
3. **点击分享按钮**（方框+箭头图标）
4. **选择"添加到主屏幕"**
5. **输入应用名称**：`Starbucks Admin`
6. **点击"添加"**

### 使用体验：

- ✅ 像原生应用一样打开

- ✅ 支持离线使用
- ✅ 支持横竖屏
- ✅ 所有功能可用
