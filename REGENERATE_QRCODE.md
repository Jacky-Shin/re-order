# 🔄 重新生成二维码指南

## 🔍 问题原因

您之前保存的二维码指向的是**旧的URL**（可能是 `localhost:3000` 或本地IP地址），现在应用已经部署到Vercel（`re-order.vercel.app`），所以旧的二维码无法访问。

## ✅ 解决方案：重新生成二维码

### 步骤1：访问二维码生成页面

在浏览器中访问：
```
https://re-order.vercel.app/qrcode
```

### 步骤2：检查基础URL

页面会自动检测并显示基础URL：
- 应该显示：`https://re-order.vercel.app`
- 如果显示的是其他URL，请手动修改为：`https://re-order.vercel.app`

### 步骤3：输入桌号（可选）

- 如果需要关联桌号，输入桌号（如：A1、1号桌等）
- 如果不输入，二维码会指向通用菜单页面

### 步骤4：生成并下载二维码

1. 二维码会自动生成并显示在页面上
2. 点击"下载二维码"按钮
3. 保存PNG图片

### 步骤5：测试二维码

1. 使用手机扫描新下载的二维码
2. 应该能正常进入菜单页面：`https://re-order.vercel.app/menu`

### 步骤6：打印并放置

1. 打印二维码
2. 放置在对应的餐桌上
3. 顾客扫描即可点餐

## 📋 二维码URL格式

生成的二维码URL格式：
- **有桌号**：`https://re-order.vercel.app/menu?table=A1`
- **无桌号**：`https://re-order.vercel.app/menu`

## ⚠️ 重要提示

1. **旧的二维码已失效**：之前保存的二维码无法使用，必须重新生成
2. **使用新的URL**：所有新生成的二维码都应该使用 `re-order.vercel.app` 域名
3. **测试后再打印**：建议先测试扫描，确认能正常访问后再批量打印

## 🎯 快速操作

1. 访问：`https://re-order.vercel.app/qrcode`
2. 确认URL是：`https://re-order.vercel.app`
3. 输入桌号（可选）
4. 下载二维码
5. 测试扫描
6. 打印使用
